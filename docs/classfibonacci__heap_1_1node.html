Detailed Description

template<typename K, typename T, typename Compare = std::less<K>>
class fibonacci_heap< K, T, Compare >::node

Reference to nodes in Fibonacci heap.

Objects of node should be returned from methods of fibonacci_heap, and will keep valid throughout the whole lifetime of the Fibonacci heap. If the original Fibonacci heap is copied to a new heap, node objects of the original Fibonacci heap will not work on the new heap.
